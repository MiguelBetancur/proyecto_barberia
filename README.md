# Sistema de Gestión para Barberías
Aplicación web para la gestión de citas, servicios, usuarios y pagos en una barbería. Desarrollada con Laravel (backend monolítico), MySQL (base de datos) y React (microservicio frontend).

## Instalación y ejecución — Microservicio React
git clone https://github.com/MiguelBetancur/microservicio-barberia.git
cd microservicio-barberia
php artisan serv
cd barberia_front
npm run dev
## Tecnologías utilizadas
- Laravel 8
- MySQL
- Bootstrap 5.3
- JavaScript (validaciones, Plotly.js, fetch API)
- React.js (microservicio)
- barryvdh/laravel-dompdf (generación de PDFs)
- JSONPlaceholder (API externa consumida)
